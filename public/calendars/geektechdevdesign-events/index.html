Geek/Tech/Dev/Design Events

This calendar will list events related to geek, tech, dev and design mainly in Singapore. It will be managed by the community for the community. Access will be readily given to the influencers of relevant user groups in Singapore (Just ask if you need access. We will approve if you are not a weirdo). Once you have access to the calendar. Just update your group’s events (or other relevant events) any time! No need to ask for permission!

Do remember to add a link to the relevant Facebook event or the google group announcement in the event description!
